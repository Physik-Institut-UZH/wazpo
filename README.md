# wazpo
Warp Zero Postprocessing (with c++)


# Requirements

 - **gcc >= ? ** (it works with 8.2)
 - **python** and **pip**
 - **cmake>=3.9**

# installation 

Install the c++ package manager:
```bash
pip install conan
```

Install dependencies and compile wazpo:
```bash
git clone ....
cd wazpo/build
source install.sh
```
